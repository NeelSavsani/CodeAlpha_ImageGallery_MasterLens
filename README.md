<h1>📸 MasterLens Pro</h1>

A modern and responsive Image Gallery Website built using HTML, CSS, and JavaScript featuring smooth animations, fullscreen preview mode, image navigation, zoom functionality, dark/light theme support, and mobile-friendly design.

🌐 Live Demo https://masterlenspro.vercel.app/
<hr>

<h3>✨ Features</h3>
<ul>
<li>🖼️ Dynamic image gallery</li>
<li>🔍 Fullscreen image preview</li>
<li>⬅️➡️ Next / Previous image navigation</li>
<li>🔎 Zoom in / Zoom out functionality</li>
<li>📱 Fully responsive mobile design</li>
<li>🍔 Mobile hamburger navigation</li>
<li>🌙 Dark & Light theme toggle</li>
<li>⚡ Smooth transitions and animations</li>
<li>⌨️ Keyboard navigation support</li>
<li>👆 Mobile swipe gesture support</li>
<li>💤 Lazy image loading for performance</li>
</ul>
<hr>

<h3>🛠️ Technologies Used</h3>
<ul>
<li>HTML5</li>
<li>CSS3</li>
<li>JavaScript (Vanilla JS)</li>
<li>Font Awesome</li>
<li>Google Fonts</li>
<li>Vercel Hosting</li>
</ul>

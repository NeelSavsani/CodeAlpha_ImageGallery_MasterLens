<h1 >📸 MasterLens Pro</h1>

<p >
  A modern and responsive <b>Image Gallery Website</b> built using 
  <b>HTML</b>, <b>CSS</b>, and <b>JavaScript</b>.
</p>

<p >
    🌐 Live Demo
  <a href="https://masterlenspro.vercel.app/" target="_blank">
    https://masterlenspro.vercel.app/
  </a>
</p>

<hr>

<h2>✨ Features</h2>

<ul>
  <li>🖼️ Dynamic image gallery</li>
  <li>🔍 Fullscreen image preview</li>
  <li>⬅️➡️ Next / Previous image navigation</li>
  <li>🔎 Zoom in & Zoom out functionality</li>
  <li>📱 Fully responsive mobile design</li>
  <li>🍔 Mobile hamburger navigation</li>
  <li>🌙 Dark & Light theme support</li>
  <li>⚡ Smooth animations and transitions</li>
  <li>⌨️ Keyboard navigation support</li>
  <li>👆 Mobile swipe gesture support</li>
  <li>💤 Lazy image loading</li>
</ul>

<hr>

<h2>🛠️ Technologies Used</h2>

<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript (Vanilla JS)</li>
  <li>Font Awesome</li>
  <li>Google Fonts</li>
  <li>Vercel Hosting</li>
</ul>
